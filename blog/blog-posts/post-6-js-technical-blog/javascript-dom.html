What is Javascript and how does it relate to HTML & CSS?

Imagine its your friends birthday and you have decided you want to make them a cake. Think of HTML as the cake. But without icing, no flavours, no colours, just a cake. Boring huh. Your best guess is that your friend isn't going to be that stoked with a flavourless, colourless cake on their birthday. CSS is your flavour, colour and decorations. Add those and you've got a pretty good looking cake. But your friend is pretty fussy. You know that this still won't cut it. Here's where Javascipt(JS) comes to save the day(birthday). With a sprinkle of JS the cake is now singing happy birthday to your friend, when they blow out the candles, the candles relight themselves. You realise you got your friends age wrong on the cake and the cake has now corrected the age for you! Javascript does all the interactions!!

Control flow & Loops

Control flow is the order in which your computer executes the code/statements in your script or js file. The code will run in order from the first line to the last line unless told otherwise. And otherwise happens a lot. Statements such as Loops and Conditionals will change and control the flow. Think of it like you're getting ready to go to work. And you can't leave without having a coffee. You would use an if else statement. if you have had your coffee - go to work. else - stay home.

Loops are similar, but you have the ability to run the code multiple times. Your shirt has 6 buttons on it. And you can't leave until 5 of them are done up. You would use a for loop. Using the for loop you would run the code such as - do up button - 5 times over.

What is the DOM?

Click the button above to see an example of the DOM in action.

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects; that way, programming languages can interact with the page.

Accessing data from Arrays and Objects

You access an array element by referring to the index number: 

const cars = ["Saab", "Volvo", "BMW"];
            let car = cars[0];
Note: Array indexes start with 0. [0] is the first element. [1] is the second element.

Accessing data from ojects is done etiher by using .dot notation or brackets []. 

objectName.propertyName
objectName["propertyName"]
