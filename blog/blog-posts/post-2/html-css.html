What's the difference?

Relative, absolute & fixed positoning

29 June 2022

Relative

Naturally an element set to relative will be position itself on the page after or "relative to" the element before it. When used with additional properties such as; left: 20px; top: 20px; . The element will now position itself 20px from the left and 20px from the top of its natural position. When the element moves, no other element on the layout will be affected.

Absolute

Absolute positioning gives you the freedom to place your element precisely where you want it. Elements will position themselves relative to the first relatively or absolutely positioned parent element. If there are none the element will position related directly to the HTML element(the page).

Fixed

An element with the position fixed, will be positioned relative to the viewport. When the page is scrolled the element will stay in the same place. All other elements will ignore the fixed element and it does leave a gap in the page. Very commonly used or the Navigation Bar or Menu.

Summary

Understanding the position property is a key part to mastering css layouts. With practice you will learn to know when it is right to use each position value. Check out the tutorial on w3schools to learn more.
